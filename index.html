Zooniverse Incubator Front-Ends

IFE Projects

Each front-end implements a custom workflow for data collection on the Zooniverse citizen science platform, tailored to the specific needs of its science domain.

Cosmic Canvas

Brush-tool classifier for galaxy morphology

A drawing-based classifier built for annotation tasks where volunteers paint over features in astronomical images using a configurable brush tool — identifying tidal tails, star-forming regions, and other morphological structures in galaxy imagery.

Author Hayley Roberts (@astrohayley)
React Vite Canvas API Panoptes API
GitHub

Juno UVS Aurora Viewer

Mapping Jupiter's ultraviolet aurora

An interactive map viewer for Juno UVS (Ultraviolet Spectrograph) data. Displays UV photon counts from NASA's Juno spacecraft on a Leaflet map using Jupiter's native north polar stereographic projection. Volunteers click to place measurement points that capture geographic coordinates and UV intensity readings.

Author Ramanakumar Sankar (@ramanakumars)
React TypeScript Vite Leaflet FastAPI GeoTIFF
GitHub

Solar Jet Hunter: Velocity

Measuring solar jet speeds from time-distance plots

A Streamlit-based annotation tool for the Solar Jet Hunter citizen science project. Volunteers draw lines on time-distance diagrams of solar jets observed by NASA's SDO/AIA instrument, enabling measurement of jet velocities in the solar chromosphere and corona.

Author Sophie Musset (@somusset)
Python Streamlit Docker Panoptes API
Live App GitHub

ZooIFE

AI-assisted image segmentation for Zooniverse

An interactive classification interface that integrates Meta's SAM2 (Segment Anything Model) with the Zooniverse platform. Volunteers use point, freehand, and brush tools to highlight regions of interest, with SAM2 providing real-time segmentation assistance.

Author Kameswara Bharadwaj Mantha (@AgentM-GEG)
